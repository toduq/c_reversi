# c_reversi

C言語を思い出すために書いてみたリバーシです。

自分が黒、自分が白、コンピュータ同士で対戦して100戦して結果を見る、の３種類が実装されています。
アルファベータ法のようなアルゴリズムで、7手先まで読みますが弱いです。

役には立たないと思いますが、ライセンスはCC0です。ご自由にお使いください。（ソースコードが間違っていても責任は取れません。）
